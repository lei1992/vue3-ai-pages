想打造一个个人博客，记录一些技术点，以及生活点。
由于不懂后端知识，目前全部使用前端技术实现。

## 技术栈

- 前端：Vue3 + Vite + TypeScript

- 代码检测：ESLint + Prettier + Husky + lint-staged + stylelint

- 代码提交：Commitlint + Conventional Commits

- 路由：Vue Router

- 状态管理：Pinia

- UI 框架：Element Plus

- 样式：SCSS + Tailwind CSS

- 请求库：Axios

- 打包：Vite

## 功能

- 首页：酷炫的首页，展示博客主要模块

- 文章列表：展示文章列表

- 文章详情：展示文章内容

- 关于：展示个人信息

- 搜索：搜索文章标题

- 标签：展示文章标签

- 分类：展示文章分类

- 归档：展示文章归档

- 留言：展示留言列表
